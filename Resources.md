# OSINT Tools and Resources

## Reconnaissance
🔗 [Who.is Record](https://who.is/) - Lookup domain ownership details.<br>
🔗 [CentrelOps](https://centralops.net/co/) - Online network tools for DNS and IP analysis.<br>
🔗 [Viewdns.info](https://viewdns.info/) - Provides DNS and WHOIS tools.<br>
🔗 [DnsChecker](https://dnschecker.org/) - Check DNS propagation globally. <br>
🔗 [Whois Lookup](https://whois.domaintools.com/) - Domain WHOIS database lookup. <br>
🔗 [Mxtoolbox](https://mxtoolbox.com/) - Email, network, and DNS tools. <br>
🔗 [Mac Vendor Lookup](https://macaddress.io/) - Identify device manufacturers by MAC address. <br>
🔗 [DNS Dumpster](https://dnsdumpster.com/) - Visualize DNS records for domains. <br>
🔗 [Check SSL/TLS Certificate](https://crt.sh/) - Search public SSL/TLS certificates. <br>
🔗 [DNSlytics](https://dnslytics.com/) - Analyze DNS and domain details. <br>
